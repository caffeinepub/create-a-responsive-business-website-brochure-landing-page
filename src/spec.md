# Specification

## Summary
**Goal:** Build a responsive single-page business brochure website with a consistent warm neutral theme, section navigation, a validated contact form, and basic SEO/accessibility.

**Planned changes:**
- Create a single-page layout with navbar, hero (headline/subheadline/CTA), About, Services (3+), Testimonials (3+), Contact (placeholders + form), and footer (social placeholders).
- Add in-page navigation (anchor links/scroll targets) to jump to each section.
- Apply a coherent visual theme (colors/typography/spacing/components) that avoids a blue/purple-dominant primary palette.
- Implement a contact form (name, email, message) with client-side validation and a no-backend submission confirmation state.
- Add basic SEO metadata (title + meta description) and accessible markup (heading hierarchy, focus states, sufficient contrast).
- Generate and place required images under `frontend/public/assets/generated` and use them (logo, hero banner, service icons).

**User-visible outcome:** Visitors can browse a polished, mobile-friendly business landing page, jump between sections from the navbar, view services/testimonials, and submit the contact form to see a clear “thank you” confirmation—without any backend integration.
